# Velocite - Платформа управления проектами

![Preview](https://cdn.poehali.dev/templates/nexus-work-management-ru/preview-ru.jpg)

Современный лендинг для SaaS-платформы управления проектами и командной работой. Темная тема с минималистичным дизайном. Включает hero-секцию с призывом к действию, бегущую строку с логотипами партнеров, bento-grid с функциями, блок тарифов и футер.
